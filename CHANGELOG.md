# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-06-21

### Added
- **Core Kernel**: `SYSTEM_CORE.md` defining the 5 non-negotiable laws, state machine, and error recovery.
- **Skill-Based Architecture**: 3-tier hierarchy using dynamic Agent Skills to prevent context window collapse.
- **6 Operational Phases**: Mission Synthesis, Execution Engine, Verification Matrix, Cognitive Persistence, Interface Protocols, and Adaptive Protocols.
- **16 Skill Protocols**: Detailed, actionable guidelines for requirement distillation, state checkpointing, pentagonal audit, self-correction, and more.
- **Anti-Rationalization Safeguards**: Built into every skill to anticipate and rebut agent excuses.
- **Examples**: Integration templates for Cursor, Claude Code, GitHub Copilot, and Gemini CLI / Antigravity.
- **Templates**: Standardized formats for `PLAN.md`, `SNAPSHOT_*.md`, and `progress_log.md`.
- **Install Script**: `install.sh` for bootstrapping projects via curl.
- **Documentation**: `QUICKSTART.md`, `CHEATSHEET.md`, and `CONTEXT_MANAGEMENT.md`.
