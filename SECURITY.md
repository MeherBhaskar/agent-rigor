# Security Policy

## Supported Versions

Agent Rigor is a framework of Agent Skills and guidelines. We provide security updates for the current major version.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within the Agent Rigor protocols (such as a prompt injection risk or a pattern that causes an agent to act destructively without escalating), please report it by opening a **GitHub Security Advisory** on this repository.

Do not report security vulnerabilities through public GitHub issues.

We will review the report and respond as quickly as possible, usually within 48 hours.
